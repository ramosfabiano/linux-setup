# Linux Setup

Post-install setup scripts for personal use.


## Debian 13 (Trixie)

`sudo bash -c "$(wget -qO- https://raw.githubusercontent.com/ramosfabiano/linux-setup/main/debian-13-setup.sh) | tee log.txt"`

## AlmaLinux 10
